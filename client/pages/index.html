Dette Projekt

Dette proejkt demonstrerer en moderne web applikation af Joe & The Juice som system. Projektet kombinerer Joe & The Juices signaturstil med moderne webteknologier.

Tekniske Funktioner:

  • PM2 og Nginx

  • Twilio One-Time-Password

  • Sikker brugergodkendelse

  • Oversigt af Joe & The Juice menu

  • Tilmelding af velkomstbrev

Applikationen er bygget via:

  • HTML5 & CSS for sstruktur og stil

  • JavaScript for dynamisk funktionalitet

  • Node.js backend med Express

  • SQLite database til data opbevaring
